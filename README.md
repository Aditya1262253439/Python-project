# Python-project
This project analyses transactional e-commerce data to extract actionable business insights through an interactive analytics dashboard, demonstrating time-based trends, customer segmentation, conversion bottlenecks, regional performance, and festival impacts.

Business Impact
Without systematic analysis, businesses face:
•	Inefficient inventory management
•	Suboptimal marketing spend
•	Lost revenue from unaddressed conversion issues
•	Missed geographic opportunities
•	Reactive rather than proactive planning


Solution Approach
This project addresses these challenges through:
•	Time series analysis for demand forecasting
•	RFM segmentation for targeted marketing
•	Funnel analysis for conversion optimization
•	Geographic visualization for market insights
•	Festival impact analysis for seasonal planning
•	Interactive Streamlet dashboard for self-service analytics
